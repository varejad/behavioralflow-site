Sobre o Projeto BehavioralFlow

🔍 O que é o BehavioralFlow?

BehavioralFlow é um projeto que simula agentes comportamentais interagindo em um ambiente visual. A proposta é aplicar conceitos da análise do comportamento (como reforçamento, molduras relacionais e seleção por consequências) em uma interface visual programada com p5.js.

🕒 Última atualização

24 de maio de 2025

📁 Estrutura atual do projeto

  • Frontend com HTML, CSS e p5.js (Canvas)
  • Backend em Python, hospedado no Render
  • Hospedagem do site via GitHub Pages

✅ Funcionalidades atuais

  • Visualização de múltiplos agentes
  • Movimentação aleatória no canvas

🚀 Planejamento futuro

  • Receber dados reais do backend via fetch()
  • Visualizar relações entre agentes
  • Destacar reforços, punições e outros eventos comportamentais
  • Implementar agentes com molduras relacionais e inferência
